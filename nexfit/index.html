Top Set

1 seul set à l'effort cible. Repos complet avant (3–5 min). Note la charge et les reps. La progression se mesure là.

Back-Off

–10 à –20% selon la semaine. RPE toujours 1 point en dessous du TS. Volume d'accumulation sans cramer le SNC.

Planning semaine

Ex : Lun (A) · Mar (B) · Jeu (C) · Sam (D). 48h min entre séances. S4 : +24h avant chaque PR.

Reset cycle

S5 = retour RPE 7 avec les charges PR de S4 comme nouvelle base. Si PR réussi : +2.5 kg. Recommencer le cycle.
